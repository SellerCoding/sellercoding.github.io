In loading

Pada saat loading, Anda diminta untuk mengisi nama Anda. Keep calm saja, nama itu bersifat PRIVAT dan tidak akan dikirim ke pembuat blog dan disimpan sebagai cookie

Kelebihannya, jika dengan cookie, nama yang sudah diketik tidak perlu diketik ulang lagi dan langsung tersimpan di dalam browser

The parts

Bagian atas terdapat toolbar. Di situ Anda bisa menjelajah. Sebenarnya, di bagian itu ada bagian yang tersembunyi, yaitu sebuah papan informasi untuk memberitahu informasi, alert, dan warning

Klik bagian chat untuk membuka buku tamu

Untuk membuka menu, geser posisi mouse ke sebelah kiri. Ada banyak pilihan untuk dipilih

Di toolbar, ada 2 tombol anak panah yang bisa dibuka. Bagian kiri berisi 'perkakas' untuk mengakses beberapa fitur di blog kami, sedangkan di bagian kanan hanyalah untuk sebuah informasi

Untuk membuka bagian-bagian tertentu melalui action overflow, klik kanan. Untuk mengetahui tentang action overflow, buka Basic Uses dan klik Action Overflow

In the menu section

Di menu, ada 3 pilihan, search, navigation, dan ads. Namun untuk ads masih sedikit dulu

Bagian search adalah tempat untuk melakukan pencarian di blog ini

Bagian navigation adalah home, FAQ, About this blog, Kode warna HTML untuk mengecek kode-kode warna dalam HTML, Disclaimer yang berisi tentang disclaimer blog kami, lalu ada link exchange untuk bertukar link

Blog kami memang masih valid HTML 5, tetapi tidak semuanya. Untuk berlangganan RSS klik RSS saja

WARNING

PERHATIAN:

  • Blog kami diproteksi oleh Anti Pl4g1at v3.5 yang sudah mensupport berbagai macam sistem operasi, mulai dari Windows, Linux, hingga Mac.   • Blog kami memang mempunyai Anti Klik Kanan, tapi tidak seperti pada blog kebanyakan yang menggunakan gambar, melainkan dengan action overflow. Menyalahgunakannya menjadi alat plagiat sangat tidak diperkenankan. Kami tidak terima dengan hal ini

'Ganti warna background'

Untuk mengganti warna background, Anda tak perlu khawatir karena pilihan warnanya sedikit. Di blog kami, pilihan warnanya sangat banyak tergantung Anda

Caranya, geser ketiga slider tersebut. Slider pertama untuk mengatur warna merah, slider kedua untuk warna hijau, dan slider ketiga untuk warna biru. Kami akan terus mengembangkannya sehingga value dapat diatur dengan cara ketik dan geser slider

Cara penggunaannya mirip dengan Anda menggunakan pengatur warna di Microsoft Office. Seperti Anda ketahui, di Microsoft Office juga menerapkan cara yang sama dan kemudian saya adopsi di blog kami

Jika Anda ingin mereset warnanya ke settingan semula, tekan tombol reset. Khusus untuk hari-hari tertentu, background ini dapat diganti secara khusus hanya dengan menekan tombol paling kiri (Tergantung tanggal berapa). Untuk menutupnya, tekan tombol close

U-Ad Remover (Khusus pengguna atau mantan pengguna Speedy)

Merasa terganggu dengan iklan yang diinject oleh telkom dan menutupi bagian atas dari halaman blog kami? Kami telah membuat tool yang bisa menghilangkan iklan ini

Blog kami kalau bisa akan kami iklankan oleh layanan iklan Google AdWords dan tidak akan diiklankan oleh iklannya telkom yang dipasang secara inject, karena untuk memasang iklan, harus minta izin terhadap penggunanya

Parahnya, iklan ini tidak ada tombol closenya seperti kamu melihat iklan sejenis dan melayang di beberapa website. Jadi bagaimana cara menghilangkan iklannya selain melalui Inspect Element? Kami pun menerapkan fungsi U-Ad Remover spesial untuk langganan TelkomSpeedy

Blog kami memang sudah diterapkan fungsi "cfs.u-ad.info remover". Fungsi ini akan menghapus iklan secara otomatis pada saat halaman blog lagi diload. Dalam beberapa kasus, iklan memang masih tetap muncul karena pada saat keload, sistem javascript tidak mengenali adanya iklan. Jika ini masih terjadi, Anda bisa klik kanan, lalu klik remove ads. Kami akan mengembangkan teknologi ini agar iklan tidak muncul selamanya

HTML Code Conversion

Widget ini adalah widget yang berada di komentar. Terima kasih kepada halaman ini

Scriptnya saja yang kita ambil, sisanya kita pakai elemen jQuery UI aja.

Cara penggunaan:

  1. Klik . Tombolnya terletak di bagian atas kolom komentar:
  2. Lalu akan muncul dialog HTML Conversion Code. Ketikkan kode kamu yang ingin dikonversikan
  3. Setelah mengeik kodenya, di bagian bawahnya terdapat opsi untuk mematikan atau menyalakan beberapa fitur konversi, misalnya mengubah & menjadi &amp;, dan sejenisnya. Anda bisa mematikan atau menyalakan sesuka kalian
  4. Setelah itu, klik konversi

    Informasi: Setelah klik konversi, kode itu tidak bisa diubah kembali. Untuk mengubah kode, silahkan klik tombol bersihkan. Tombol bersihkan akan menghapus seluruh field yang ada dan meresetnya ke kondisi default

  5. Setelah itu, salin kode tersebut ke kolom komentar. Caranya, tinggal copy, lalu klik bersihkan, lalu klik tombol tutup untuk menutup dialog. Setelah itu, salin ke kolom komentar

    PERHATIAN: Tombol bersihkan akan menghapus seluruh field yang ada dan meresetnya ke kondisi default. Pastikan Anda mempunyai backupnya terlebih dahulu sebelum mengklik tombol bersihkan

Action overflow

Action overflow adalah tempat-tempat untuk bersemanyamnya beberapa pilihan. Anda bisa kembali ke beranda, melakukan pencarian, membookmark (browser tertentu saja), mengcopy (Tetap pakai ctrl+c atau command+c), Open Link in New Tab (Tetap pakai ctrl+click atau command+click), dan melakukan reload

Selain itu, Anda bisa menyembunyikan kotak melayang, kembali ke atas atau menuju ke bawah, membuka buku tamu, backlink, print halaman, membuka kotak informasi, menghilangkan iklan, dan mengaktifkan energy saving mode

Untuk mengaktifkannya, Anda bisa mengklik kanan di halaman manapun dan muncul kotak ini dari bawah halaman

Informasi: Mengklik kanan pada halaman blog atau menekan tombol menu (antara alt dan ctrl serta terletak di bagian kanan bawah keyboard) akan mengaktifkan fungsi Anti Klik Kanan dan action overflow

Search

Jika Anda ingin untuk mencari sesuatu di blog kami, Anda bisa mencarinya melalui Action overflow atau dari navigation menu yang bisa diakses dengan menggeser cursor ke bagian kiri dari halaman ini

Setelah muncul, ketik keyword yang Anda inginkan lalu tekan tombol enter. Maka keyword yang Anda cari langsung ditemukan

Fullscreen

Merasa terganggu oleh kotak melayang? Anda bisa mengaktifkan fungsi Full Screen pada blog. Caranya dengan menekan tombol F11 (FN+F11 untuk FN Terbalik atau pengguna Mac), Command+Control+F, atau Command+Shift+F (Pengguna Google Chrome di Mac), atau Anda bisa mengklik kanan -> Full Screen

Dengan menekan keempat parameter di atas, semua elemen kotak melayang akan hilang kecuali buku tamu yang memang sengaja administrator diamkan agar Anda bisa melakukan chat terhadap admin

PERHATIAN: Menekan tombol F11, Command+Ctrl+F, atau Command+Shift+F berkali-kali mungkin dapat menyebabkan malfungsi di sistem jQuery

Guest Book

Buku tamu adalah tempat para pengunjung blog untuk mengobrol. Anda bisa menyampaikan pesan Anda, kesan mengunjung blog ini, atau menyampaikan komentar jika komentar dikunci

PERHATIAN: Untuk mencegah pengguna yang lain kesal dan menimbulkan konflik serta menimbulkan spam di buku tamu, kami sarankan untuk membaca tatacara menyampaikan pesan melalui link etika. Jika tatacara ini dilanggar, komentar mungkin akan dihapus dalam waktu beberapa hari saja

Untuk membuka buku tamu, Anda klik saja di tombol chat dan buku tamu pun keluar. Atau bisa juga klik kanan -> Buku Tamu, atau bisa juga melalui navigation menu -> GuestBook

Isilah Nama Anda di bagian name. Kalau tidak mau isi nama, ketik sajalah Anonymous. Di bagian email/url, isilah url. Ketikkan pesanmu di message

PERHATIAN: Untuk privasi, kamu menyarankan untuk menggunakan url saja ketimbang menggunakan email. Jika tidak punya satu url pun, kosongkan saja


Informasi: Karakter dalam pesan maksimal 100 karakter

Untuk mengirim pesan, ketik enter atau klik Go
